# 🎵 EmoTune - Emotion Based Music Recommendation System

## 📌 Project Overview

EmoTune is an AI-based music recommendation web application that detects user emotions through facial expressions and recommends suitable songs from YouTube based on the detected mood.

This project was developed using computer vision and deep learning techniques.


## 🛠 Technologies Used

- Python
- TensorFlow
- OpenCV
- MediaPipe
- Streamlit


## ✨ Features

- 😊 Real-time emotion detection using facial expressions
- 🤖 AI-based emotion recognition
- 🎵 YouTube music recommendations based on mood
- 🌐 Interactive web application

## 🚀 How It Works

1. Captures user facial expressions through camera
2. Detects emotion using AI model
3. Matches emotion with suitable songs
4. Provides YouTube music recommendations
